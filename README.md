Data Classification Using AI

Project Overview

This project demonstrates a basic Machine Learning classification model using the Decision Tree Algorithm.

The model is trained using the Iris Dataset and predicts the category of a flower based on input measurements.

---

Objective

To build a simple AI classification system that:

- Loads and understands a dataset
- Splits data into training and testing sets
- Trains a Decision Tree model
- Predicts the class of new data
- Evaluates model accuracy

---

Technologies Used

- Python
- Scikit-learn
- Machine Learning
- Decision Tree Classifier

---

Dataset

Iris Dataset

Features:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Classes:

- Setosa
- Versicolor
- Virginica

---

Project Workflow

Load Dataset
↓
Split Data
↓
Train Model
↓
Predict Class
↓
Evaluate Accuracy
